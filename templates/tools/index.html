Admin tools

Operational tasks run against the live OpenStack services as the signed-in Keystone user. Listings are read straight from the regional database — fast and cheap; state-changing actions call the service API and need a Keystone-scoped session.

L3 routers

Reschedule Neutron virtual routers between L3 agents — drain a network node before taking it out of service for maintenance.

VLANs

Create a provider VLAN network for a project. The admin creates the network; the project's own users add the subnet.

VLAN list

Every VLAN network in a region, with its owning project, domain, physnet, segmentation id, and state. Read-only.

Ports in BUILD

Every Neutron port stuck in BUILD across all regions — quick triage for hung port creations.

Networks
